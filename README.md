# 🔥 Fusion 360 Katana (Nichirin Inspired)

A custom-designed katana modeled in Fusion 360, inspired by Tanjiro Kamado's Nichirin sword from Demon Slayer.

## 🧠 About the Project
This is my first Fusion 360 project where I designed a katana as a **single-piece model**.  
The goal was to explore CAD design, precision modeling, and aesthetic detailing.

## ✨ Features
- 🗡️ Single-piece katana design  
- 🎨 Inspired by anime (Nichirin sword style)  
- 🧩 Clean and minimal geometry  
- ⚙️ Designed fully in Fusion 360  

## 🛠️ Files Included
- `.f3d` → Original Fusion 360 file  
- `.step` → Interoperable CAD format  
- `.stl` → Ready for 3D printing
